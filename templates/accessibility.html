WELLnest Accessibility Statement

WELLnest is committed to providing a website that is accessible to all users, including those with disabilities. We endeavor to follow those standards used by the Federal government for technology accessibility for people with disabilities (Section 508), and the Web Content Accessibility Guidelines (WCAG) 2.0 developed by the World Wide Web Consortium (W3C). WebMD's web portal site is regularly tested using leading web accessibility technologies and we make accessibility updates frequently.

For more information about the Federal standards, please visit the Section 508 web site https://www.section508.gov/content/learn/standards or the Federal Access Board web site https://www.access-board.gov. Additionally, for more information about the WCAG 2.0 Standards, please visit their web site https://www.w3.org/TR/WCAG20/.

In addition, some documents on WELLnest web site are produced in Portable Document Format (PDF). Efforts are currently underway to ensure that these files are produced using the latest version of Acrobat (file conversion software for PDFs). This is the most accessible technology currently available for these types of files.
